### Validation

Aadhaar

- Required
- 12 digits

OTP

- Required
- 6 digits

PAN

Regex

[A-Z]{5}[0-9]{4}[A-Z]

Validate while typing.

Use React Hook Form with Zod for all client-side validation.

--------------------------------------------

### Form Behaviour

The UI should read schema.json and automatically render:

- Labels
- Inputs
- Dropdowns
- Buttons
- Required marks
- Placeholders
- Validation rules

Do not hardcode any field names or validation rules.

If any property is missing from schema.json, gracefully handle it without crashing.

### Architecture Rules

1. Functional components only.

2. Hooks only.

3. Reusable components.

4. Keep business logic outside UI components.

5. Separate validation logic.

6. Separate API layer.

7. Never duplicate code.

8. Keep components small.

9. Clean imports.

10. Proper folder organization.

11. Keep components reusable and configurable through props.

12. Use Context API only for shared global state.

13. Keep API calls inside the services layer.

14. Follow clean architecture and separation of concerns.
