Create a production-grade PostgreSQL database layer for the Udyam Registration Assignment.

### Tech Stack

- PostgreSQL
- Prisma ORM
- Prisma Migrate
- Prisma Client
- dotenv

--------------------------------------------------

### Goal

Design and implement a production-ready relational database for the first two steps of the Udyam Registration Portal.

The database should store all registration information collected from:

1. Aadhaar Verification
2. OTP Verification
3. PAN Verification

The database should only support the first two steps of the Udyam Registration process.

Do not design tables for later registration steps unless required for future extensibility.

The generated Prisma schema should integrate directly with the Express backend without requiring manual modifications.

Do NOT generate backend controllers, frontend code, APIs, Docker, or deployment.

Only generate the database layer.
